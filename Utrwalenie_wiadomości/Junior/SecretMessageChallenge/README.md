# Secret Message Challenge (GitHub)

To jest mini-zadanie do lekcji o GitHub (tylko przeglądarka) + uważności w analizie zmian.

## Zadanie dla ucznia
1. Wejdź w plik `dziennik.txt` na GitHub.
2. Kliknij **History** (Historia pliku).
3. Otwieraj kolejne commity i sprawdzaj **diff** (co się zmieniło).
4. W każdym commicie znajdziesz **jedno dopisane słowo** na końcu linii zaczynającej się od `WSKAZÓWKA:`.
5. Zapisz dopisane słowa w kolejności commitów i złóż z nich ukrytą wiadomość.

## Bonus
- Zrób screen lub notatkę: w którym commicie pojawiło się które słowo?
- Spróbuj znaleźć zmianę, która wygląda na „zwykłą poprawkę”, ale jednak jest wskazówką.
